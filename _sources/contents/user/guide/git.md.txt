# Git

## What is Git?

Git is a distributed version-control system for tracking changes in software development projects.
It can also be used with any general purpose directory of files, as such, in the lab is also used
to keep track of the changes when writting a manuscript. Then, no matter you are going to work
locally with git, or you are going to interact with github repositories, you have define your name
and email.

## Your user parameters

Once you are going to use git, be sure that your user name and email is defined in your local file
`~/.giconfig`. If not, please execute the following commands with your info:


```bash
$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com
```
